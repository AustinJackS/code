How to turn background on most sites any color with javascript bookmark:


Add a bookmark with this a a url: javascript:var color = prompt('color');document.body.style.background = color;void 0;


How turn turn background into an image:



Create a bookamrk with this url: javascript:var im = prompt('image url');document.body.style = 'background: url('+im+') no-repeat center center fixed; -webkit-background-size: cover; -moz-background-size: cover; -o-background-size: cover; background-size: cover;';void 0
